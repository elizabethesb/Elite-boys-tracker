─────────────────────────────────────────────────────────────────────────────
TROUBLESHOOTING
─────────────────────────────────────────────────────────────────────────────

Can't deploy?
  → Make sure you committed files to GitHub before importing to Vercel
  → Check GitHub repo shows all files (not empty)

Won't install as app?
  → Must use HTTPS (Vercel provides this automatically)
  → Safari on iPhone, Chrome on Android
  → Wait 30 seconds after first visit before trying to install

Cloud sync not working?
  → Check Supabase credentials in Vercel env vars
  → Make sure Supabase table was created (run the SQL)
  → Open browser console (F12 → Console) to see errors

Data not showing?
  → Check localStorage: F12 → Application → Local Storage
  → Should see eb_points, eb_logs, eb_tasks

─────────────────────────────────────────────────────────────────────────────
WHAT'S HAPPENING BEHIND THE SCENES
─────────────────────────────────────────────────────────────────────────────

Web App (React + Vite):
  • Fast, modern UI built with React
  • Vite bundles it for deployment
  • Runs entirely in browser (no server calls by default)

PWA (Progressive Web App):
  • Service worker caches files → works offline
  • Manifest.json makes it installable
  • Acts like native app on home screen
  • Data syncs when you go online

Cloud Sync (Supabase):
  • Optional: only if you set it up
  • Parents sign in with email
  • Data encrypted in transit and at rest
  • Only you can see your family's data
